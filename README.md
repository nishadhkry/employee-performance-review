# Employee Performance Review Analysis Using PySpark

## Project Overview

This project demonstrates how to perform employee performance review analysis using PySpark DataFrame operations.

The project focuses on:

- Filtering high-performing employees
- Handling missing values
- Removing duplicate records
- Aggregating department-wise ratings
- Joining multiple DataFrames
- Performing SQL queries
- Using window functions for trend analysis

This case study simulates a real-world HR analytics workflow used in enterprise data engineering and big data environments.

---

# Technologies Used

- Apache Spark (PySpark)
- Python
- Spark SQL
- DataFrame API
- Window Functions
